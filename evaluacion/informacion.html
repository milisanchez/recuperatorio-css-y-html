ACERCA DE HTML ...

¿QUE ES HTML?

  • Es un Lenguaje de marcado de hipertexto (Hyper Text Markup Language)
  • No se encarga del aspecto visual de la página
  • Es un lenguaje de estructura (no de programación)
  • Es la base de todas las páginas web

¿COMO CREO MI ARCHIVO?

Primero debemos crear un archivo y llamarlo index.html, luego al presionar el signo "!", aparecera la estructura del documento html y podremos empezar a codear nuestra pagina web.

ETIQUETAS

  • HEADER: sirve como una introduccion, alli podemos encontrar elementos del encabezado, logo, formulario de busqueda, nombre del autor, entre otros
  • BODY: tiene el contenido del documento html y solamente puede haber uno.
  • FOOTER: es el pie de pagina, puede tener informacion acerca del autor de la pagina asi como tambien los derechos del autor o enlaces a documentos.
  • SECTION: funciona como un contenedor donde esta el contenido relacionado entre si, sirve para crear un bloque que a su vez tiene mas bloques dentro
  • ARTICLE: funciona como un contenedor pero es independiente. En cualquier lugar del codigo en el que se encuentre va a tener sentido.
  • ASIDE: se usa para expresar contenido que no es parte del tema de la pagina, como anuncios

ATRIBUTOS

Son valores adicionales que configuran a los elementos y ajustan su comportamiento.

  • Class: se usa para darle estilos a nuestra página a través de CSS
  • ID: es un identificador que se utiliza para navegar a traves de anclas
  • Data:Nos permite guardar algún valor en la etiqueta HTML
  • Title: muestra una descripción del elemento al q pertenece

ENLACES

Conectan una página con otras. Utilizan obligatoriamente estos atributos:

  • Href: donde escribimos la dirección de la pagina que queremos.
  • Target:configura como queremos visualizar el sitio hacia donde nos dirigimos. Se establece con la etiqueta que viene de ancla

LISTAS

Sirve para ordenar y enumerar datos.

  • ul -> unordered list: listas desordenadas
  • ol -> ordered list: listas ordenadas.
  • dl -> definition list: Se utilizan para hacer una lista de definiciones

TABLAS

Antes se utilizaban para mostrar el contenido de una pagina web. Actualmente esto ya no se utiliza, ya que ahora se usa el CSS para el aspecto visual de la pagina.

ESTRUCTURA DE UNA TABLA
  • table → etiqueta que encierra una tabla
  • tr → table row, etiqueta que sirve para construir una fila, la cantidad d celdas va a definir la cantidad d columna de la tabla.
  • td → table data, etiqueta q construye una celda
ATRIBUTOS DE UNA TABLA
  • rowspan: hace q la celda ocupe mas de una fila
  • colspan: hace q la celda ocupe mas de una columna.
  • colgroup: permite seleccionar una columna especifica

FORMULARIOS

Nos permiten mostrar o ingresar información.

ESTRUCTURA
  • Form → es la etiqueta q contiene nuestro formulario
  • Label → permite escribir el nombre del campo a rellenar.
  • input → permite qué el usuario ingrese información
  • button → crea un boton

CONTENIDO EMBEBIDO

Es todo el contenido externo que pesa más en cuanto a tamaño y se añaden a un sitio web, como imágenes, videos, audios e iframes.
